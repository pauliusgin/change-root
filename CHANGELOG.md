# Change Log

All notable changes to the "change-root" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.2.0]

- Changed icon.

## [0.1.0]

- Added `change-root.setSelectionAsRoot` command (Explorer context menu + keybinding) to reopen the window with a selected folder as the workspace root.
- Added `change-root.setParentAsRoot` command to move the workspace root up one directory.
- Default keybindings (active while the Explorer is focused): `.` sets the selected folder as root, `backspace` sets the parent as root.
